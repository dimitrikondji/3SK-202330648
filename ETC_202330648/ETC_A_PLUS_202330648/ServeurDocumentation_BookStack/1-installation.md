# Installation – Serveur de documentation BookStack

## Contexte
BookStack est utilisé pour centraliser la documentation technique du réseau.

## Environnement
- OS : Ubuntu Server 24.04 LTS
- IP : 192.168.16.12
- Accès DNS : doc.202330648.lan
## Technologies utilisées
- Docker
- Docker Compose
- BookStack
- MySQL
- Nginx (reverse proxy)

## Procédure
1. Installation de Docker et Docker Compose
2. Création du fichier docker-compose.yml
3. Déploiement avec docker compose up -d
4. Accès au service via navigateur

Conclusion :
Le service BookStack est installé et accessible.
