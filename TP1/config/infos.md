# Informations sur la configuration

## Emplacements des fichiers sensibles
- `/etc/php/8.1/apache2/php.ini` → configuration PHP
- `/etc/apache2/sites-available/` → configurations des vhosts disponibles
- `/etc/apache2/sites-enabled/` → configurations actives
- `/var/www/deepweb.com/` → fichiers du site WordPress
- `/var/www/forum.deepweb.com/` → fichiers du forum phpBB
